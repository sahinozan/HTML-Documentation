Introduction

  • HTML stands for Hyper Text Markup Language.

  • HTML is the standard markup language for creating Web pages.

  • HTML describes the structure of a Web page.

  • HTML elements tell the browser how to display the content.

Elements

  • The HTML element is everything from the start tag to the end tag.

  • There are some examples below.

Links

  • HTML links are hyperlinks.

  • You can click on a link and jump to another document.

  • When you move the mouse over a link, the mouse arrow will turn into a little hand.

href Attribute

  • The most important attribute of the element is the href attribute, which indicates the link's destination.

  • The link text is the part that will be visible to the reader.

  • Clicking on the link text, will send the reader to the specified URL address.

Attribute Syntax Result
href <a href="https://www.ieu.edu.tr" alt="Izmir University of Economics">www.ieu.edu.tr</a> www.ieu.edu.tr

target Attribute

■ The target attribute specifies where to open the linked document.

Attribute Syntax
target <a href="https://www.ieu.edu.tr/" target="_blank">Visit IEU!</a>
  • _self - Default. Opens the document in the same window/tab as it was clicked
  • _blank - Opens the document in a new window or tab.
  • _parent - Opens the document in the parent frame.
  • _top - Opens the document in the full body of the window

Images

  • Images can improve the design and the appearance of a web page.

  • The HTML <img> tag is used to embed an image in a web page.

Syntax Example
<img src="ieu_logo.jpg" alt="Izmir University of Economics">
<img src="amazon.png" alt="Amazon">

src Attribute

  • src - Specifies the path to the image.

  • alt - Specifies an alternate text for the image.

Animated Images

Syntax Result
<img src=amogus.gif alt="Amongus GIF" style="width: 100px; height: 100px;">

Image Size - Width and Height

Attribute Syntax Result
style <img src="google.png" alt="Google" style="width:100px;height:100px;">
width and height <img src="google.png" alt="Google" width="200" height="200">

Image as a Link

■ To use an image as a link, put the <img> tag inside the <a> tag.

Syntax Result
<a href="https://www.ieu.edu.tr" target="_blank> <img src="ieu_logo.jpg" alt="Izmir University of Economics" style="width:70px;height:70px;"></a>
